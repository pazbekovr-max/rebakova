# Book Tracker

## Автор
Ребякова Рада Ильинична

## Описание
Приложение «Book Tracker» — это графический интерфейс для управления списком прочитанных книг. Программа позволяет добавлять книги с указанием автора, жанра и количества страниц, фильтровать список по критериям и сохранять данные в формате JSON.

## Функционал
- Добавление новых книг с валидацией данных.
- Фильтрация по жанру и минимальному количеству страниц.
- Сохранение и загрузка базы данных в файл `books.json`.
- Удаление выбранных записей.
